# Style Examples

This directory is for local author-style reference snippets used by the writer prompt.

Before running the Iteration 009 true-model smoke, choose 3-5 short passages from your local normalized source texts and save them as Markdown files in this directory. Recommended mix:

- 1-2 passages for opening-chapter rhythm.
- 1-2 passages for restrained environment or action description.
- 1 passage for character interiority, preferably close to Lu Mingfei's point of view.

Each passage should be roughly 200-500 Chinese characters. Keep only the examples you want the model to imitate for rhythm, restraint, imagery, and dialogue cadence.

Do not commit original copyrighted passages. The real example files should stay local under `data/style_examples/*.md`; this README is only an instruction placeholder.
